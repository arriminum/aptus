Filter

Use the grayscale filter for a subtle hover effect, common in team or logo grids. Hover the image to see it.

Zoom

Wrap an image in .img-zoom-wrap to smoothly zoom it in on hover, without overflowing its container. Hover the image to see it.

  <img src="docs/assets/example.jpg" alt="Example image">
</a>

Caption

Wrap an image and a .img-caption element in .img-caption-wrap to overlay a caption at the bottom of the image.

<div class="img-caption-wrap">
  <img src="docs/assets/example.jpg" alt="Example image">
  <p class="img-caption">Photo caption goes here</p>
</div>

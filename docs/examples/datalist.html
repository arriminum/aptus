Aptus Datalist CSS

Styling for tabular data fetched from a service: sortable-looking headers, hoverable rows, row actions, and a summary/pager footer.

Basic table

.datalist-table on a <table>. Rows highlight on hover by default. This example sets --datalist-table-border: 1px solid #dddddd for the outer border and body-row separators.

Nome Tipo Criação Último acesso
John Smith Manager 10/07/2026
Alice Doe Analyst 02/03/2026 14/07/2026
Bruno Alves Analyst 21/01/2026 15/07/2026 
<table class="datalist-table">
  <thead>
    <tr>
      <th width="44%">Nome</th>
      <th width="24%">Tipo</th>
      <th width="16%">Criação</th>
      <th width="16%">Último acesso</th>
    </tr>
  </thead>
  <tbody>
    <tr data-id="16">
      <td>John Smith</td>
      <td>Manager</td>
      <td>10/07/2026</td>
      <td></td>
    </tr>
    <tr data-id="17" class="datalist-row-alt">
      ...
    </tr>
  </tbody>
</table>

Disabling row hover

Add .datalist-no-hover to .datalist-table when rows are not clickable.

Nome Tipo
John Smith Manager 
<table class="datalist-table datalist-no-hover">...</table>

Row actions

.datalist-action-link for inline links/commands inside a cell, such as edit or delete.

Nome Ações
John Smith ✎Editar 
<a href="#" class="datalist-action-link"><i>✎</i>Editar</a>

Empty state

.datalist-empty for a "no results" row or panel below the table.

Nome Tipo
Nenhum registro encontrado.
<div class="datalist-empty">Nenhum registro encontrado.</div>

Summary and pager

.datalist-summary wraps a record count and a .datalist-pager list; the current page uses .datalist-pager-current instead of a link.

Total de registros: 3
  • «
  • 1
  • 2
  • »
<div class="datalist-summary">
  <span>Total de registros: 3</span>
  <ul class="datalist-pager">
    <li><a href="#">«</a></li>
    <li><span class="datalist-pager-current">1</span></li>
    <li><a href="#">2</a></li>
    <li><a href="#">»</a></li>
  </ul>
</div>

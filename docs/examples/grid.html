Aptus Grid CSS

12-column, float-based grid. Columns stack full-width below 768px and take their fractional width at and above it. Resize the window to see the responsive behavior.

Basic row

A .g-row with equal .col6 columns.

.col6
.col6
<div class="g-row">
  <div class="col6">First column</div>
  <div class="col6">Second column</div>
</div>

Unequal columns

Columns don't need to be equal; any combination that adds up to 12 works.

.col4
.col4
.col4
.col3
.col9
<div class="g-row">
  <div class="col4">...</div>
  <div class="col4">...</div>
  <div class="col4">...</div>
</div>

<div class="g-row">
  <div class="col3">...</div>
  <div class="col9">...</div>
</div>

Gutter

Add .gutter to .g-row to space columns apart horizontally, instead of having them sit flush against each other.

.col4
.col4
.col4
<div class="g-row gutter">
  <div class="col4">...</div>
  <div class="col4">...</div>
  <div class="col4">...</div>
</div>

Offset

Use .col-offset1–.col-offset11 on a column to push it to the right by that many columns, leaving empty space instead of a placeholder column.

.col6 .col-offset3
.col4
.col4 .col-offset4
<div class="g-row">
  <div class="col6 col-offset3">Centered column</div>
</div>

<div class="g-row">
  <div class="col4">...</div>
  <div class="col4 col-offset4">Pushed further right</div>
</div>

Gutter and offset combined

.gutter and .col-offset* can be combined on the same row.

.col3 .col-offset1
.col3
.col3
<div class="g-row gutter">
  <div class="col3 col-offset1">...</div>
  <div class="col3">...</div>
  <div class="col3">...</div>
</div>

Flex vertical alignment

Add .flex-align to .g-row to switch it to display: flex; align-items: center so columns of different heights align vertically. Use .align-top, .align-middle, or .align-bottom on a column to override its own alignment.

Tall column
.align-top
.align-bottom
<div class="g-row flex-align">
  <div class="col4">Tall column</div>
  <div class="col4 align-top">...</div>
  <div class="col4 align-bottom">...</div>
</div>
